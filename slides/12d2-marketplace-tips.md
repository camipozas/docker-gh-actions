# Marketplace Tips

### How to find actions

1. Go to **github.com/marketplace**
2. Filter by **Actions**
3. Search by keyword (e.g., "deploy", "lint", "docker")
4. Check the ⭐ stars and **verified creator** badge

### Best practices

- 🔒 Always use a **specific version** (`@v4`, not `@main`)
- ✅ Prefer actions with many ⭐ stars
- 📖 Read the action's **README** for required inputs
