# Other Container Registries

| Registry | When to use |
| --- | --- |
| **Docker Hub** | Public images, open source |
| **GitHub Container Registry (ghcr.io)** | Integrated with GitHub Actions |
| **AWS ECR** | Private registry, AWS projects |
| **Google Artifact Registry** | Private registry, GCP projects |

ghcr.io is free for public repos and works great with GitHub Actions.
